FAQ

Questions, answered straight

Can I sign up right now?

Not at the moment. This build is archived and isn't creating new accounts — the demo above shows exactly what the product does.

Do I need any other editing software?

No. Aiditor runs in the browser — you upload files, describe the edit, and it renders the result.

Can it use my existing Premiere or Photoshop files?

Yes. Aiditor reads project files from Premiere, Final Cut, and Photoshop and can continue from where you left off.

What if I need something really specific?

That's the point — it writes custom code for the exact edit you describe, instead of relying on a fixed set of presets.
